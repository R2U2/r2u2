# Instructions for generating Vivado Project:
1. Open Vivado
2. In the Tcl console, locate this directory (i.e., R2U2_HW)
3. Run the following command in the Tcl console: source R2U2_HW.tcl
4. Replace "PATH_TO_DIRECTORY" in interface/sim/interface_tb.vhd and ftMonitor/sim/testbench/ft_monitor_tb.vhd to your own absolute directory path.
