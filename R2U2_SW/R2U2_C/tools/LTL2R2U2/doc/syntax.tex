\subsection{Explicit Mapping of Variables and Subformulas}
Atomic variables are mapped to a specific memory address using the
{\tt map} directive. If not explicitely mapped, addresses are automatically
allocated and a warning is issued.

Sub-formulas can have labels. Memory addresses corresponding to the
labels are provided in the mapping file.
\begin{verbatim}
	% set logic -- currently in there; will be deprecated
	%
logic(ft).

	% map named variables to specific addresses
	% all other variables (atomics) get indices starting with 0,1,..
map foobar = 99.
map xx = 12.

	% subformulas can be labeled, such that they can be extracted
	% for BN interface
L0003: (((X ((! ((F[0,8] foobar -> foo )))))) & 
	(((a0 U[20, 34] foobar2)) | (xx))) &
f2:( xx | f1: ~ foobar) &
(xx | foobar).
\end{verbatim}
